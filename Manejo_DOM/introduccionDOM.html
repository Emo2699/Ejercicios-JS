DOM: Documment Object Model o Documento de Modelo de Objeto

El DOM es la forma en como se representa codigo de HTML y CSS para trabajarlo con JS, es una interfaz estandarizada. para mas info ir a MDN

BOM: Browser Object Model

No se le considera una API estandar, puesto que cada navegador tiene sus propias caracteristicas y las trabajan a su propio estilo, sin embargo, ya se comienza a estandarizar algunas cosas. Es por ello que antes la biblioteca JQuery era muy popular ya que era la forma de interaccion con diversos navegadores.
