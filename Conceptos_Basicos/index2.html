  • Modulos Import y Export

Para importar archivos js a un html se hace con la etiqueta script, agregando el parametro src, donde colocamos la ruta del archivo a importar

Ademas, para que ese archivo tenga la posibilidad de importar más archivos JS por si mismo, agregamos en la etiqueta script el parametro type="module" Con esto ya especificamos que el archivo que carga el HTML sera modulo y pordra importar otros archivos

NOTA: Todos los archivos JS que importe el HTML deben de ir antes de la etiqueta de cierre del body
